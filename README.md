# Dtour Cloud

Dtour Cloud frontend — landing page, token page, vault UI, and branding assets.

Whitelabel overlay for ElizaCloud. Powered by ElizaOS + ElizaCloud infrastructure.

## Structure

```
public/brand/dtour/     # Brand assets — logos, video bg, SVGs
src/pages/              # Landing page, token page, login
src/dashboard/vault/    # Vault staking UI
src/lib/                # Branding constants
src/middleware/          # Tier gate middleware
wrangler-dtour.toml     # Cloudflare Workers config
```
